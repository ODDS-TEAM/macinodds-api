# Install Golang

## OS X

##### Install brew

Install brew from [brew.sh](http://brew.sh/)

##### Install Git

```
$ brew install git 
```

##### Install Go

Install golang binaries using `brew`

```
$ brew install go
```

##### Testing it all

```
$ go env
$ go version
```

##### Pull package

```
$ go get
```

##### Run

```
$ go run server.go
```
